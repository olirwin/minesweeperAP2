graphicalboard module¶

Ce module fournit essentiellement une fonction de création d’une fenêtre graphique pour jouer au démineur. Il nécessite les modules cell et minesweeper.

  • Création d’une fenêtre de jeu :

    graphicalboard.create(game)[source]¶

    This function creates the graphical board from a game. It also launches the event loop. Thus, this is the only function to run to have a functional graphical board.

    Parameters:game (Minesweeper) – the minesweeper game
    Returns:None

  • Gestion des évènements (clics) :

    graphicalboard.__changestate(board, game, x, y)[source]¶

    This function is called on left-click on a button.

    Parameters:
    • board (list of list of button) – the board of buttons
    • game – the minesweeper game
    • x (int) – the x-coordinate of the cell
    • y (int) – the y-coordinate of the cell
    graphicalboard.__changeflag(evt, board, game, x, y)[source]¶

    This function is called on right-click on a button.

    Parameters:
    • board (list of list of button) – the board of buttons
    • game (Minesweeper) – the minesweeper game
    • x (int) – the x-coordinate of the cell
    • y (int) – the y-coordinate of the cell
    graphicalboard.__redraw(board, game, x, y)[source]¶

    This function draws the board. Positions x and y are used to test which bomb icon has to be drawn.

    Parameters:
    • board (list of list of button) – the board of buttons
    • game (game) – the minesweeper game
    • x (int) – the x-coordinate of the cell
    • y (int) – the y-coordinate of the cell

  • Gestion de la partie (gagnant / perdant) :

    graphicalboard.__test_end(board, game)[source]¶

    This function tests if the game is finished or not. In the first case, depending on the state of the game, all graphical cells are diabled or events are unbinded.

    Parameters:
    • board (list of list of button) – the board of buttons
    • game (Minesweeper) – the minesweeper game
    graphicalboard.__block_game(board, game)[source]¶

    This function is called once the player wins. The chosen behavior is to let the board as it and to unbind events.

    Parameters:
    • board (list of list of button) – the board of buttons
    • game (Minesweeper) – the minesweeper game
    graphicalboard.__disable_game(board, game)[source]¶

    This function is called once the player looses. The chosen behavior is to shade the board and to unbind events.

    Parameters:
    • board (list of list of button) – the board of buttons
    • game (Minesweeper) – the minesweeper game
