console_main module¶

Ce module fournit essentiellement une fonction de création d’une partie de démineur en console. Il nécessite les modules cell et minesweeper.

  • Création d’une partie :

    console_main.main()[source]¶

    Main function to play a console version of the minesweeper game

  • Fonctions auxilliaires :

    console_main.game_manual()[source]¶

    The manual for the minesweeper game. Opens and reads into the console one or two files, a welcome file (manual.txt) and if needed a help file (howtoplay.txt)

    Returns:None
    Side effect:reads files
    UC:none
    console_main.print_game(game)[source]¶

    Prints the console output of the game

    Parameters:game – the game to print
    Returns:none
    Side effect:prints a grid in terminal
    UC:none
    console_main.apply_action(x, y, action, game)[source]¶

    Applies the parameter action to the game cell (x, y)

    Parameters:
    • x (int) – x-coordinate of cell to play on
    • y (int) – y-coordinate of cell to play on
    • action (str) – action to apply (“R”, “S” or “U”)
    • game (Minesweeper) – the game we’re playing
    Returns:

    none
    Side effect:

    changes the state of the cell (x, y) with the action action
